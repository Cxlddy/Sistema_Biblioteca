# Sistema de Biblioteca em Python

Sistema simples de gerenciamento de biblioteca com interface gráfica.

## Tecnologias

- Python
- CustomTkinter
- Programação Orientada a Objetos

## Funcionalidades

- Cadastro de usuários
- Login
- Adicionar livros
- Remover livros
- Listar livros
- Empréstimo de livros
- Devolução de livros

## Como executar

pip install -r requirements.txt
python main.py
